# Library Plus

Library Plus is a comprehensive, full-stack library management system designed to streamline both user and administrative workflows. 

## Features

- **Book Catalog & Discovery:** Browse, search, and filter available books.
- **Borrowing & Reservations:** Manage current rentals, handle late fees, and support waitlists for unavailable books.
- **Reviews & Ratings:** Users can rate and review books they have returned.
- **Administrative Dashboard:** Role-based access control (RBAC) allows administrators to manage clients, handle inventory, review book returns, and track platform analytics.
- **Tech Stack:** 
  - **Frontend:** Next.js / React
  - **Backend:** .NET / C#
  - **Databases:** PostgreSQL (Relational Data) & MongoDB (Document Data / Event Sourcing)
  - **Containerization:** Docker & Docker Compose
